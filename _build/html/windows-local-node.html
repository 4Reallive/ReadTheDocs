Create A Windows Local Node¶

This guide will explain how to run a local node on your Windows device.

Downloading Files¶

The first step to get your node started is to download the newest client release versioin here: https://github.com/CelestialCash/Celestial/releases/. Select the Windows.zip file and it should download it. Then just unzip the files.

Starting The Node¶

All you have to do to start the node is double click the Celestiald executable and then wait for it to say successfully synced. Make sure it has been synced before you use other features like the wallet or miner.

Using The Node¶

To use your node with the wallet and miner all you have to do is make sure you dont have --daemon-address or --remote-daemon in your batch file or the command you use to start the executables.
